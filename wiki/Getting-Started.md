# Getting Started

## Installation

### Pre-built Release
1. Download the latest `.exe` from the [Releases](../../releases) page
2. Run `BackgroundCropper.exe` — no installation needed

### Build from Source
See [Building from Source](Building-from-Source).

## First Use

### 1. Select a Source Folder
Click the folder button at the top-left and choose a directory containing your images. Supported formats: JPEG, PNG, BMP, WebP.

### 2. Browse Images
Thumbnails appear in the sidebar with a polaroid-style layout. Scroll through them — already-cropped images show a green checkmark.

### 3. Select an Image
Click a thumbnail. The image appears in the main canvas with a crop overlay locked to your screen's aspect ratio.

### 4. Adjust the Crop
- **Drag** the selection to move it
- **Drag corner handles** to resize (ratio stays locked)
- **Shift+Drag** to snap to image edges
- The **rule of thirds grid** helps with composition

### 5. Set Output Folder
Click the output folder button and choose where cropped images should be saved.

### 6. Crop
Click the **Crop** button at the bottom. The image is saved to your output folder at the highest possible resolution from the original.

## Language
Switch language via the dropdown in the toolbar (NL, EN, DE, FR). The setting is remembered between sessions.
